# Keyboard Cheat Sheet

Developer workstation keybindings for the Zellij → Neovim stack.
Works reliably through any terminal chain (Windows Terminal → SSH → Zellij → Neovim).

---

## Zellij (Alt prefix)

Alt keys are captured by Zellij before reaching Neovim — no conflicts.

| Key | Action |
|-----|--------|
| `Alt+n` | New pane (split right) |
| `Alt+Shift+N` | New pane (split down) |
| `Alt+t` | New tab |
| `Alt+h/j/k/l` | Focus pane left / down / up / right |
| `Alt+1–5` | Jump to tab 1–5 |
| `Alt+z` | Zoom pane (fullscreen toggle) |
| `Alt+x` | Close pane |
| `Alt+,` | Rename tab |
| `Alt+d` | Detach session |
| `Alt+[` | Scroll mode (navigate with j/k, Ctrl+f/b) |
| `Alt+r` | Resize mode (resize with h/j/k/l) |
| `Alt+s` | Session manager |

### Ctrl shortcuts (reliable through SSH)

| Key | Action | Vim equivalent |

## Language-specific notes

### Python
- LSP: **pyright**
- Debug: **debugpy** via `F5` — uses `pytest` as test runner
- Format: **black** / **ruff** (via conform.nvim)

### C++
- LSP: **clangd** (system package, uses `compile_commands.json`)
- Debug: **codelldb** via `F5` — prompts for executable path
- Format: **clang-format**

### Rust
- LSP: **rust-analyzer** (via rustup component)
- Debug: **codelldb** with Rust pretty-printers via `F5`
- Format: **rustfmt** (`<leader>lf`)
